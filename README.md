# dogNpup

Project Set-Up

```bash
npm install
```

Run the App

```bash
ionic serve
```

Project structure to follow:

```
-src
    -app
        app.module.ts
        app.component.ts
        app.routing.module.ts
        ...
        - modules
            	- module1
               		- components
              		- pages
              		module1.service.ts
               		module1.module.ts
               		module1.routes.ts

            	- module2
               		- components
               		- pages
               		module2.service.ts
               		module2.module.ts
               		module2.routes.ts
        - shared
           	- components
           	- mocks
           	- models
           	- ...
```
